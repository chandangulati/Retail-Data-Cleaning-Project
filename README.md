# Retail Data Cleaning Project

## Objective

The objective of this project is to clean and preprocess a retail transaction dataset using Python and Pandas.

## Tasks Performed

* Missing value handling
* Duplicate record removal
* Data cleaning and preprocessing
* Feature engineering (Total_Sales column)
* Export cleaned data to CSV

## Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Deliverables

* retail_data_cleaning.ipynb
* cleaned_retail_transactions.csv
* Data_Cleaning_Report.docx

## Outcome

A cleaned retail dataset was generated and exported for further analysis.
